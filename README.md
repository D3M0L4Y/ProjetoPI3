## 🎯 Sobre o Projeto
Este sistema foi desenvolvido para o **Projeto Integrador III (2026)**, com foco em criar uma solução full-stack para gestão imobiliária, integrando análise de dados automatizada e recursos rigorosos de acessibilidade digital.

### 🛠️ Tecnologias Utilizadas
- **Back-end:** Python com **FastAPI** (Hospedado no Render)
- **Data Science:** Biblioteca **Pandas** para processamento de estatísticas em tempo real
- **Banco de Dados:** **PostgreSQL** (Gerenciado via Supabase)
- **Front-end:** HTML5, CSS3 e JavaScript (Hospedado no GitHub Pages)
- **Integração:** SQLAlchemy e CORSMiddleware para comunicação segura

## 🚀 Funcionalidades Principais
- **Consumo de API:** O front-end consome dados dinâmicos via Fetch API.
- **Análise Estatística:** Rota `/analise` que processa média de preços e distribuição por cidade via Pandas.
- **Acessibilidade Completa:** Implementação de VLibras, Alto Contraste e tags semânticas para leitores de tela.
- **Arquitetura em Nuvem:** Deploy contínuo (CI/CD) integrado ao GitHub.

## 📁 Estrutura do Repositório
- `/docs`: Contém o front-end estático (GitHub Pages).
- `main.py`: Script principal da API FastAPI.
- `requirements.txt`: Dependências do ambiente Python.
- `.env.example`: Modelo para variáveis de ambiente do banco de dados.
